# Basic Printer Control

Printerhive gives you practical control over connected printers from the web interface.

## Available controls

Available controls depend on the printer type and firmware, but may include:

- start a print,
- pause, resume, or cancel a print,
- set nozzle and bed temperatures,
- control fans,
- run calibration actions,
- execute Klipper macros,
- view and use camera streams,
- skip objects when supported,
- manage AMS or external filament selection.

## Before using remote control

Remote control is convenient, but it still controls a real machine. Before starting a job remotely, verify:

1. the printer is clear and ready,
2. the correct plate is installed,
3. filament is loaded correctly,
4. the selected file matches the printer and nozzle,
5. the printer can run safely unattended.
