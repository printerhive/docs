# Scheduled Prints

Scheduled prints let you prepare a job now and start it later.

## When to use scheduled prints

Use scheduling when:

- a print should start after working hours,
- the printer should begin after another task,
- you want to prepare several jobs in advance,
- the operator who prepares work is not the person who starts it.

## Managing scheduled jobs

When sending a job, choose a future start time if scheduling is available for the selected printer and workflow.

Before confirming, check:

- selected printer,
- selected plate,
- filament and AMS tray,
- quantity,
- preheating settings,
- whether the printer will be safe to run unattended.

## Editing and cancelling

You can modify scheduled jobs before they start. If you cancel a scheduled job, Printerhive returns it to the queue and turns off preheating that was enabled specifically for that job.
